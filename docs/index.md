# Macintosh SE/30 Modern SDK

Welcome to the future of the past. This SDK allows you to build modern C++ applications for System 7.5 using a containerized workflow.

## Features

* **Retro68 Toolchain**: GCC-based cross-compilation for m68k.
* **Modern C++17**: Write clean, modern code.
* **Declarative GUI**: No more `WaitNextEvent` loops or `.r` files.
* **Secure Networking**: Connect to the modern web via the Bridge Server.
